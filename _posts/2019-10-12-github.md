このウィキは GitHub Pages を使っており、編集が難しいと思います。<br />
なのでアカウントの作成方法、記事の編集の仕方を説明しようと思います。<br />
使い方が分かる人は読み飛ばしても構いません。<br />

## GitHub アカウントの作成
このウィキを編集する場合は GitHub のアカウントを作らなければなりません。<br />
以下でのやり方は全てパソコンからアクセスした場合になります。

### GitHub とは？
簡単に言うと、GitHub はプログラマの間で人気のクラウドストレージの様なWebサービスです。<br />
もっと詳しく言うと、gitと呼ばれるバージョン管理システム<br />
で作られるリポジトリをネット上に公開するサービスです。<br />
バージョン管理システムとは雑に言えば、ファイルの変更を分かり易く管理するツールです。<br />
リポジトリとは管理するフォルダ(ディレクトリ)の単位の事です。<br />
因みに、gitと GitHub は開発者も提供元も異なります。<br />
GitHub がgitと呼ばれるツールを使っているだけです。

### GitHub Pages、Jekyll とは？
GitHub Pages とは GitHub のリポジトリに保存されているファイルを、<br />
Webページとして公開するサービスの名称です。<br />
Jekyll とは簡単に言うとリポジトリ内のファイルを GitHub Pages で公開できる様に、<br />
ファイルを変換するソフトの名称です。

### ユーザー名、メールアドレス、パスワードの設定
まずは、[https://github.com](https://github.com)にアクセスしてください。<br />
すると以下の様なページが表示されます。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github00.png" width="600" /><br />
`Username` の下に<u>公開してもいいユーザー名</u>を、`Email` の下にメールアドレスを、<br />
`Password` の下にパスワードを記入してください。<br />
そしたら緑色の「**Sign up for GitHub**」というボタンを押してください。

### 認証
次に以下の様なページが表示されます。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github01.png" width="600" /><br />
チェックボックスが2つありますが、チェックしなくていいでしょう。<br />
もしかしたら以下の様に表示されるかもしれません。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github01a.png" width="600" /><br />
その場合は「検証開始」ボタンをクリックして指示に従ってください。<br />
そしたら青色の「**Next: Select a Plan**」というボタンを押してください。

### プランの選択
次に以下の様にどのプランを使用するか聞かれます。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github02.png" width="600" /><br />
金を払わなくても、つまり、無料版でも支障はないので、左側の「Free $0 USD」を選びましょう。<br />
青色の「**Choose Free**」ボタンを押したら次に進めます。

### アンケート
プランを選択し終わったら以下の様なアンケートが始まります。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github03.png" width="600" /><br />
別に答えなくても構いません。答えたい人は答えてください。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github03a.png" width="600" /><br />
一番下までスクロールすると、「**Skip this step**」を選べます。<br />
アンケートに答えなかった人はそのリンクをクリックしてください。<br />
答えた人は青色の「**Complete setup**」というボタンを押してください。

### メール認証
最後に以下の様にメールアドレスが正しいか認証する必要があります。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github04.png" width="600" /><br />
自分のメールボックスを開いて GitHub からメールが来てるか確認してください。<br />
来ていなかったら迷惑メールを探してみてください。<br />
メールを開いて青色の「**Verify email address**」というボタンを押してください。<br />
ログインが必要な場合は、最初に入力したユーザー名とパスワードを使います。<br />
ページが移行し正しくできた場合はアカウントの作成完了です。

## 記事の編集・作成
アカウントの作成が終わったら記事を編集/作成する事ができます。<br />
ただし、記事を編集/作成してもこのウィキの管理者が認証するまで反映されません。

### 編集・作成ボタンを押す
まずは、このウィキの適当なページを開いてください。<br />
以下の例では[トップページ](https://takym.github.io/nyuworld/README.html)を開いています。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github05.png" width="600" /><br />
真ん中の「**編集**」リンクをクリックするとそのページを編集できます。<br />
その隣の「**新規作成**」リンクをクリックすると新しい記事を書けます。

### 編集
編集をクリックした場合は以下の様にページの編集ができます。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github06.png" width="600" /><br />
Markdown形式で編集します。<br />
Markdownが詳しく分からない方は[こちら(外部に飛びます)](https://qiita.com/Minalinsky_1911/items/b684cfabe0f2fde0c67b)等を参考にしてください。<br />
ページの先頭には必ず以下の様な文字列を入れてください。<br />
Jekyll のシステム上これがないといけません。<br />

```markdown
---
title: ページの題名
authors:
  - 作成者のユーザー名1
  - 作成者のユーザー名2
  - 作成者のユーザー名3
  - ...
---
```

<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github06a.png" width="600" /><br />
編集し終わったら、ページの下側の `Commit Changes` (`Propose file change` と表示される事もあります)<br />
の下にある `Update <ファイル名>` を何処をどの様に編集したのか記入してください。<br />
緑色の「**Commit Changes**」ボタンまたは「**Propose file change**」ボタンを押せば編集が完了します。

### 新規作成
新規作成をクリックした場合は以下の様にページの作成ができます。<br />
<img src="https://takym.github.io/nyuworld/articles/2019-10-12-github07.png" width="600" /><br />
基本的に編集する場合と同じです。<br />
ただし、ファイル名等を決めないといけません。<br />
ファイル名の後ろには必ず「.md」を付けてください。(つまり拡張子を「.md」にします)<br />
作成し終わったら、ページの下側の `Commit New File` (`Propose File Change` と表示される事もあります)<br />
の下にある `Create new file` を何を何故作成したのか記入してください。<br />
緑色の「**Commit new file**」ボタンまたは「**Propose new file**」ボタンを押せば作成が完了します。
